Python3教程

Python是跨平台的，它可以运行在Windows、Linux/Unix和Mac系统上。

在开始学习Python之前，首先需要安装Python的运行环境。

Linux

如果你使用Linux，作者认为你应该具有一定的使用经验。

主流的Linux发行版都自带了Python。如果你使用的是Arch Linux的话，那么，默认的Python解释器版本就是Python3。

在Debian、Ubuntu等发行版中，你可以使用Python命令进入Python2的交互环境，使用Python3命令来进入Python3的交互环境。它们对应的包管理命令是pip和pip3。

使用下面的命令安装pip：

sudo apt-get install python-pip python3-pip

如果你使用的是CentOS这一类发行版，你需要编译安装Python3。为了确保标准库完整可用，建议你在编译前安装openssl等系统库。

如果你不知道怎么安装，下面是一些提示：

wget
tar zxvf
cd
./configure
make && make install

你可以使用Python的easy_install来安装pip。

如果你仍然不知道如何安装，建议使用Windows操作系统。

Mac OS X

在Python官网上找到相应的版本并安装。请点击这里下载并安装。
