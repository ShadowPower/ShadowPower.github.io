'设置apm镜像，解决Atom插件下载慢的问题'

由于某些原因，在国内安装Atom插件速度非常缓慢。实际上Atom使用的插件管理器apm和npm是一样的东西，可以通过指定npm镜像来加快Atom插件安装速度，提升Atom插件安装成功率。

然而我在网上搜了一下，发现没有相关的教程来教你如何设置apm镜像。于是，我决定写下本篇教程，为大家节省宝贵的时间。

要怎么做呢？

首先，Atom的配置文件保存在 用户目录/.atom 文件夹中。Windows中的用户目录默认为 C:\User\用户名 ，Linux/Mac则为 /home/用户名 。在npm里，我们可以通过编辑 .npmrc 文件来配置npm。同样，我们也可以编辑 .apmrc 文件来配置apm。我们只需要一项配置：

registry = https://registry.npm.taobao.org

将这一行保存为 .apmrc 文件，放在 用户目录/.atom 目录中，我们就可以通过淘宝npm镜像来安装Atom插件了。为了获得加速效果，我们需要通过 apm install 插件名 这条命令来安装。当然，你可以用Python之类的语言编写一个插件批量安装脚本。
